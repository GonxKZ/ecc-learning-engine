Quick Start Guide

1

Clone and Build

bash 
git clone https://github.com/your-repo/ecscope.git
cd ecscope && mkdir build && cd build
cmake .. && make
2

Run Memory Laboratory

bash 
./ecscope_performance_lab_console

This launches the interactive memory laboratory where you can experiment with different allocation strategies and see their performance impact in real-time.

3

Launch Full GUI (Optional)

bash 
# Install SDL2
sudo apt install libsdl2-dev libgl1-mesa-dev  # Ubuntu/Debian
brew install sdl2  # macOS

# Build with graphics support
cmake -DECSCOPE_ENABLE_GRAPHICS=ON ..
make && ./ecscope_ui

Verify Your Installation
